## 自定义mc服务器列表

不支持基岩版，请窒息

第一次尝试写插件，如有意见可以反馈orz

### 目前支持指令

#### mc

指令总支，后续会陆续添加别的功能

#### mc.list

查看当前已添加服务器列表

#### mc.add [ip]  [description]

添加服务器

#### mc.remove [ip]

移除列表内服务器

#### mc.ping [ip]

请求服务器参数
ping服务器后返回服务器图标
